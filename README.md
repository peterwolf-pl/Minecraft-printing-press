# Gutenberg Printing Press

Educational Fabric mod for Minecraft 26.1.2 that builds an oversized, interactive Gutenberg-style printing press.

## In-game flow

1. Place the Gutenberg Printing Press block. It builds an 11 x 9 x 8 voxel press around the master block, with a wooden frame, bed, drawer, platen, and screw.
2. Right-click the press with a Movable Type Composition. By default it prints `GUTENBERG PRESS`. Rename the composition in an anvil to use a short custom text.
3. Right-click with Printing Ink or an Ink Roller to ink the raised type.
4. Right-click with a Blank Paper Sheet to place paper on the bed.
5. Right-click with an empty hand to push the drawer under the platen.
6. Right-click with an empty hand again to tighten the screw.
7. Right-click with an empty hand once more to press. The platen lowers briefly.
8. When printing finishes, right-click with an empty hand to remove the Printed Paper Sheet.

The Printed Paper Sheet stores its text in item custom data and shows the printed text in its tooltip.

## Notes

This is intentionally manual and visual. It is not an automation machine: every historical step is a separate player interaction with chat feedback.
